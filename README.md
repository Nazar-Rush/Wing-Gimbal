# Wing-Gimbal

This project is for me to put a gimbal with a camera onto an fixed wing that I have in order to fly around and track vehicles. This will hopefully record some rocket engine hot fires.

---

## Hardware

\*\*Pixhawk FC
-> running ardupilot
-> https://www.amazon.com/Controller-Wireless-Telemetry-Quadcopter-Multirotor/dp/B07NRMFTXL/

\*\*RPI 4B
-> running Raspberry PI OS Lite
-> used for giving instructions to the camera gimbal and recording the video

\*\*FPV Transmitter
-> used for sending video signal back to the ground station to ensure proper targeting
